DOM Traversal

Traversing the DOM with JavaScript

Be able to move from one place in the HTML document (the DOM - Document Object Model) to another is a key skill to have when working with JavaScript that is part of a web page or web application.

We can traverse down the DOM, which means go in to nested child elements from your current location or whe can traverse up the DOM, which means selecting parent or ancestor elements from our current location. We can also traverse on the same level by move through an elements siblings.

Moving Up the DOM

We can move up the DOM using the using the parentNode property for direct parents or the closet() method for find ancestors (parents, grandparents, etc...).
