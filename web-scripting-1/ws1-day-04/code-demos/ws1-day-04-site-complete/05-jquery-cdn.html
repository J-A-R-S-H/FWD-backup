jQuery - CDN

Including jQuery via a CDN

A CDN is a way of hosting a file in multiple places around the world to improve the download performance of files.

jQuery is a popular library and it is hosted in many places. The most common place to get jQuery is from jQuery's own CDN.

Usually you will want to choose the latest minified production version of jQuery.

Don't use a CDN if you think you will be developing offline.
