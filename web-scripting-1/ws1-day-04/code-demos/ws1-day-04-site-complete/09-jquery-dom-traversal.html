jQuery DOM Traversal

Traversing the DOM with jQuery

jQuery was built for traversing the DOM. Compared to native JavaScript, traversing the DOM with jQuery is a breeze.

With jQuery we can very easily navigate to the next element, the previous element, the parent, all the parents, the closest ancesstor, any child element and any descendant element.

View the jQuery docs for DOM Traversing.

parent()

Select the child box's parent when the child box is clicked on.

closest()

Select the child box's first Ancestor that matches a selector using closest().

parents()

Select all the child box's ancestor that matches a selector using parents().
