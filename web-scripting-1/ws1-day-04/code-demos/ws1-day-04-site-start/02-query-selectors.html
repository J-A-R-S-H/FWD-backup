Query DOM Selectors

The following selectors are static and are not automatically updated if the DOM is changed.

JavaScript Query Selector Methods

  • document.querySelector()
    • Selects the frist item that matches a CSS selector
  • document.querySelectorAll()
    • Selects all the elements that match a CSS selector
    • Returns a NodeList
    • The forEach() loop is available on NodeLists
