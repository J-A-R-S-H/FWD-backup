DOM Manipulation

Manipulating the DOM with JavaScript

Changing the DOM with newly aquired data or due to events is a common requirement for many scripts.

JavaScript provides many methods for cloning, deleting, appending, prepending, inserting before and after elements and changing the text content or HTML of an element.

innerHTML vs TextContent

textContent returns on the "text" of an element and strips out the HTML and innerHTML returns all content including the HTML tags. In some contexts, textContent can be safer as it can prevent XSS attacks

Other DOM Manipulation

  • Clone Element - cloneNode()
  • Remove - remove()
  • Append - append() or appendChild()
  • Prepend - prepend()
  • Insert Before - insertBefore()
  • Insert Adjacent - insertAdjacentElement()
  • Replace - replaceWith()
