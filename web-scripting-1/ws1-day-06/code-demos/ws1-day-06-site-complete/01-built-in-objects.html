Built-In Objects

Almost Everything is an Object in JavaScript

Almost everything in JavaScript is an object. The only entities that are not considered objects in JavaScript are primitive values. Primitives include strings, numbers, booleans, null and undefined.

Built-In Objects

The JavaScript that runs in the browsers creates a couple of handy built in objects.

  • window
  • document

These objects provide the developer with useful properties about the browser such as window.innerHeigh and window.innerWidth which output the size of the content area of the viewport. The document object provides many useful methods for selecting elements on the page such as the document.getElementById() which selects and returns an HTML element with an id passed into the method call.
