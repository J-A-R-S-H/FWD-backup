JSON - Introduction

About JSON

JSON stands for JavaScript Object Notation. It is based on JavaScript's object syntax with a few additional syntax rules. It is a common data format used to transmit data between computer. It is an open, plain text and human readable format.

JSON is a popular format used in JavaScript web applications.

The major syntax differences between JavaScript's regular object syntax and JSON are JSON's keys must be wrapped in double quotes and comments are not allowed.

JSON data can be and often is wrapped in an array "[]".

JSON values can be strings, numbers, booleans and null. Numbers, Booleans and null should not be wrapped in double quotes. String values should always be wrapped in double quotes.

Parsing JSON Data

Since JSON is very similar to a JavaScript object you can parse JSON data very easily with JavaScript using JavaScript's dot syntax (object.key).
