Validating a Form with HTML

With HTML5 you can provide basic form validation with out the use of JavaScript. By using the "required" attribute on input field tags the browser will require the user to fill in the field. The browser can also perform basic form validation for inputs with the type of email, to make sure the user enters a valid email address.
