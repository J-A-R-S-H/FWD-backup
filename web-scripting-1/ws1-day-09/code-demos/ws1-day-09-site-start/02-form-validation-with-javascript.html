Validating a Form with JavaScript

For more customized validation we can use JavaScript to validate our forms.

HTML5 or JavaScript form validation should never be your sole form of validation as it can easily be modified or disabled by a user. Always use server-side validation in addition to client-side validation. Server-side validation will be covered in the PHP course.
