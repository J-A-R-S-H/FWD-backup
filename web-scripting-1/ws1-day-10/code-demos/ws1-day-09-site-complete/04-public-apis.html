Public APIs

Using a Public APIs

Many social media sites such as Twitter, Instagram and Flickr offer a public API. A public API is usually a URL endpoint that you can access that will usually (note always) return JSON data.

Every company's public API is structured differently, but the good ones will have good documentation on how to access their data.
