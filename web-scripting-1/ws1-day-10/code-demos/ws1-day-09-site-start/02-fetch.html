About Fetch

Many modern web applications require making requests for data asynchronously while the user is interacting with the web page and then updating web page dynamically with the newly acquired data. This was commonly referred to as AJAX (Asynchronous JavaScript and XML) in the past. The fetch() method makes getting data asynchronously easier than the older XMLHTTPRequest() method.

Make an Asynchronous Request with Fetch()

Note: Normally we would use a web server to make a request for data. Since we do not have a web server script setup designed to return JSON data we will simply make a request to the location of our local JSON data stored inside our local data folder. In the next lesson and upcoming exercise we will look at making requests to a public API located on a public web server.
